# Automation Strategy

## Goal

```text
Provide fast confidence for important repeatable flows.
```

## Automation Focus

```text
1. E2E cases
2. Sanity cases
3. Critical regression cases
4. High-risk business flow cases
```

## Excluded From Automation

```text
- Low-priority one-time checks
- Unstable requirements
- Visual judgment-heavy cases
- Rare edge cases with high maintenance cost
```
