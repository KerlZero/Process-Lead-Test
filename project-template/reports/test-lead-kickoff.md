## QA Manual Assignment

```text
- Review requirements and backup source documents.
- Create master manual test cases.
- Validate functional behavior and user flow.
- Validate important non-functional requirements.
- Log defects with evidence.
- Mark only valuable automation candidates.
```

## QA Automation Assignment

Automation must reuse selected manual cases only.

```text
- Select E2E cases.
- Select sanity cases.
- Select critical regression cases.
- Map every script to a manual test case ID.
- Run automation and record evidence.
```
