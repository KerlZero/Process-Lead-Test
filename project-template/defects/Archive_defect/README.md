# Archive_defect

Use this folder only for defect packages that have passed retest and are closed.

Move a package here only after:

```text
1. Fix is delivered.
2. QA retest passes.
3. defect-log.md is updated.
4. defects/closed-defects.md is updated.
5. reports/quality-summary.md is updated.
```

Do not archive defects with status Open, Ready For Retest, or Retest Failed.
