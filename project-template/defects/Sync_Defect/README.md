# Sync_Defect

Use this folder for active defects that still need fixing, retesting, or Test Lead risk decision.

Create one package per defect:

```text
defects/Sync_Defect/<DEFECT-ID>/defect-log.md
defects/Sync_Defect/<DEFECT-ID>/evidence/
```

Keep the package here while the defect is:

```text
- Open
- In Fix
- Ready For Retest
- Retest Failed
- Accepted Risk
```

Do not move a package out of this folder until the retest passes or the Test Lead explicitly decides the final handling.
